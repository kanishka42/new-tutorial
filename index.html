GIT Commands:

  • git clone -Clones the branch to the your computer
  • git status -Status of the current repository, maybe on your computer
  • git add <file name> -Adds the new file to the repository/branch on your computer
  • git commit -m "some message" -Commits and locks the file to the repository/branch on your computer
  • git push -Pushes up all the changes from your repository/branch on your computer to the git (github.com) server
  • git pull -Pulls down all the changes from the git (github.com) server repository/branch to the current repository/branch on your computer
  • git branch <new branch name> -Creates a new branch
  • git checkout <branch name> -Switch to another branch
  • git checkout -b <branch name> -Creates a new branchand and switches to it
  • git branch --set-upstream-to=origin/master <branch name> -Set upstream branch "origin/master" to branch fix20
  • git push origin -Pushes the branch to the origin
  • git branch --merged -Shows merged branches
  • git git branch --no-merged -Shows un-merged branches
  • git git branch -v -Shows branches
  • git branch -d <branch name> -Deletes the specified branch
