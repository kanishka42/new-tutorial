GIT Commands:

  • git clone clones the branch to the your computer
  • git status status of the current repository, maybe on your computer
  • git add <file> adds the new file to the repository/branch on your computer
  • git commit -m "some message" commits and locks the file to the repository/branch on your computer
  • git push pushes all the changes from your repository/branch on your computer to the git (github) server
